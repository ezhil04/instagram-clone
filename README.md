INSTALLATION 

1. git clone https://github.com/ezhil04/instagram-clone.git

2. cd instagram-clone

3. npm install

4. npm start


TECH USED

1.ReactJS fire

2.Firebase

3.Material UI

FEATURES

1.Authentication with Email/Password and Google OAuth.

2.Picture uploading.

3.Comments
